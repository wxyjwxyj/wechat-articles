# Research 目录约定

## 目录结构

```
.claude/research/
├── README.md          # 本文件，结构约定
├── 00_*.md            # 汇总/索引类文档
├── 0N_*.md            # 具体研究报告（按序号递增）
└── archive/           # 超过 3 个月的旧报告移入归档
```

## 命名规范

- 格式：`NN_<主题>.md`，序号两位数，主题用英文或拼音，单词间用下划线
- 汇总/索引类固定用 `00_` 前缀
- 新报告序号在现有最大序号基础上 +1

## 内容规范

每份研究报告开头必须有：

```markdown
# 标题
> 研究日期：YYYY-MM-DD
> 研究目的：一句话说明为什么做这个研究
```

## 清理规则

- 超过 3 个月未引用的报告移入 `archive/`
- `archive/` 超过 6 个月的文件可直接删除
- `00_research_summary.md` 长期维护，不归档
