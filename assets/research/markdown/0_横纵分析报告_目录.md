## 阅读指南

**如果你完全不懂 AI**，从这里开始：

```
LLM 大模型 → Prompt Engineering → AI Agent → 特定垂直赛道（AI编程/AI搜索/AI语音/具身智能…）
```

4 篇读完，你对整个 AI 世界的全景就有了一个基础框架。剩下的可以按兴趣选读。

**如果你想系统学一遍**，按以下顺序：

```
一、基础模型 → 二、基础架构 → 三、训练与数据 → 四、推理与部署 → 五、应用范式 → 六、垂直赛道 → 七、安全与治理
```

**如果你是做应用的开发者**，只看这一条线就够了：

```
LLM大模型 → Prompt Engineering → RAG → LLM应用框架 → AI Agent
进阶：Context Engineering（上下文编排） → Agent Runtime/Harness（Agent 上线必读）
```

**如果你关注某个具体方向**，直接跳到对应类别：
- 我想了解 AI 能做什么 → **六、垂直产品赛道**
- 我想知道模型怎么训练出来的 → **三、训练与数据**
- 我想理解技术原理 → **二、基础架构**
- 我关心安全问题 → **七、安全与治理**
